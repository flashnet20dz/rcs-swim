# 🏊 نادي RCS — منظومة إدارة الاشتراكات

## التشغيل السريع

### على الكمبيوتر (ويندوز):
1. ثبّت [Node.js](https://nodejs.org)
2. انقر مرتين على `start-server.bat`
3. افتح المتصفح: `http://localhost:3000`

### على الهاتف (عبر WiFi):
1. شغّل الخادم على الكمبيوتر
2. افتح المتصفح على الهاتف
3. اكتب: `http://[IP-الكمبيوتر]:3000`
4. ثبّت كتطبيق (Add to Home Screen)

### الحسابات الافتراضية:
- المدير: `admin@rcs.dz` / `admin123`
- المدرب: `coach@rcs.dz` / `coach123`

### بناء تطبيق سطح المكتب (.exe):
```bash
npm run electron:build
```
