# 📖 دليل تثبيت نادي RCS على الكمبيوتر

## 📋 المتطلبات قبل البدء

### 1. تثبيت Node.js
- اذهب إلى: https://nodejs.org
- حمّل الإصدار **LTS** (النسخة المستقرة)
- شغّل الملف المُحمّل واتبع خطوات التثبيت
- **مهم**: اترك خيار "Add to PATH" مُفعّلاً

### 2. تثبيت Git (اختياري للتحديثات)
- اذهب إلى: https://git-scm.com
- حمّل وثبّت Git

---

## 🚀 طريقة التشغيل السريعة (الأسهل)

### الطريقة 1: ملف مضغوط (ZIP)

1. **استخرج** ملف المشروع من المجلد المضغوط
2. افتح المجلد المستخرج
3. **انقر مرتين** على `start-server.bat`
4. انتظر حتى يكتمل التثبيت (أول مرة فقط)
5. سيظهر عنوان الخادم في النافذة

### الطريقة 2: من GitHub

```bash
# افتح موجه الأوامر (CMD) أو PowerShell
git clone https://github.com/rcs-club/rcs-club-desktop.git
cd rcs-club-desktop
start-server.bat
```

---

## 💻 تشغيل التطبيق على الكمبيوتر

### بعد تشغيل `start-server.bat`:

1. ستظهر نافذة سوداء بعنوان "نادي RCS — خادم محلي"
2. انتظر حتى ترى:
   ```
   خادم RCS يعمل الآن!
   على الكمبيوتر: http://localhost:3000
   ```

3. افتح المتصفح واكتب: `http://localhost:3000`
4. ستجد صفحة تسجيل الدخول

### تسجيل الدخول:
- **المدير**: `admin@rcs.dz` / `admin123`
- **المدرب**: `coach@rcs.dz` / `coach123`

---

## 📱 ربط الهاتف بالكمبيوتر عبر WiFi

### الخطوات:

1. **تأكد أن الكمبيوتر والهاتف على نفس شبكة WiFi**

2. **شغّل الخادم** على الكمبيوتر (عن طريق `start-server.bat`)

3. **ابحث عن عنوان IP** في النافذة السوداء:
   ```
   على الهاتف (WiFi): http://192.168.1.xxx:3000
   ```

4. **على الهاتف**: افتح المتصفح واكتب العنوان

5. **ثبّت التطبيق** على الهاتف:
   - **Chrome (أندرويد)**: القائمة (⋮) → "تثبيت التطبيق"
   - **Safari (آيفون)**: زر المشاركة → "إضافة إلى الشاشة الرئيسية"

6. الآن لديك أيقونة تطبيق RCS على هاتفك!

## 🔥 جعل التطبيق يعمل كبرنامج سطح مكتب (Electron)

### بناء ملف .exe للويندوز:

```bash
# في موجه الأوامر داخل مجلد المشروع:
npm run electron:build
```

سيتم إنشاء ملف التثبيت في مجلد `dist/`:
```
dist/RCS-Club-Setup-1.0.0.exe
```

### تثبيت البرنامج:
1. شغّل `RCS-Club-Setup-1.0.0.exe`
2. اتبع خطوات التثبيت
3. ستجد أيقونة "نادي RCS" على سطح المكتب
4. انقر عليها لتشغيل البرنامج

## 🔄 التحديثات التلقائية

عند إصدار نسخة جديدة:
1. البرنامج يتحقق تلقائياً عند الإقلاع
2. يُظهر رسالة "تحديث متاح"
3. يحمّل التحديث في الخلفية
4. يثبّته عند إغلاق البرنامج

## 🛠️ استكشاف الأخطاء

### المشكلة: "Node.js not found"
**الحل**: ثبّت Node.js من https://nodejs.org

### المشكلة: "Port 3000 already in use"
**الحل**: أغلق البرامج التي تستخدم المنفذ 3000، أو غيّر المنفذ:
```bash
set PORT=3001
npm run start
```

### المشكلة: الهاتف لا يتصل
**الحلول**:
1. تأكد أن الكمبيوتر والهاتف على نفس WiFi
2. تأكد أن جدار الحماية (Firewall) يسمح بالاتصال
3. جرّب إيقاف جدار الحماية مؤقتاً
4. استخدم عنوان IP الصحيح (موجود في نافذة الخادم)

### المشكلة: قاعدة البيانات فارغة
**الحل**: شغّل أمر التهيئة:
```bash
npx prisma db push
npx tsx scripts/seed-roles.ts
```
