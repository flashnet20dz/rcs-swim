# 🏊 نادي RCS — النسخة الهجينة (محلي + سحابي)

## المعمارية

```
┌──────────────────┐     ┌──────────────────┐     ┌──────────────────┐
│  كمبيوتر (Electron)│     │  هاتف (PWA)       │     │  سحابة (Vercel)   │
│  SQLite محلي     │     │  IndexedDB محلي  │     │  Neon PostgreSQL  │
│  أوفلاين كامل    │     │  أوفلاين كامل    │     │  المصدر الموثوق   │
└────────┬─────────┘     └────────┬─────────┘     └────────┬─────────┘
         │                        │                        │
         └────────────────────────┼────────────────────────┘
                                  │
                     ┌────────────┴────────────┐
                     │  طبقة المزامنة /api/sync │
                     │  • Push: تغييراتي        │
                     │  • Pull: أحدث البيانات   │
                     │  • حل التعارضات          │
                     └─────────────────────────┘
```

## كيف يعمل

### على الهاتف (PWA)
1. افتح https://aladine-pool-manager.vercel.app على الهاتف
2. اضغط "تثبيت التطبيق" (Add to Home Screen)
3. التطبيق يصبح أيقونة على شاشتك الرئيسية
4. يعمل **أوفلاين بالكامل** — كل البيانات تُخزَّن في IndexedDB
5. عند عودة الإنترنت → مزامنة تلقائية مع السحابة

### على الكمبيوتر (Electron)
```bash
# التطوير
npm run electron:dev

# بناء نسخة Windows
npm run electron:build

# النتيجة: ملف .exe في مجلد dist/
```
- يعمل أوفلاين مع Next.js server محلي
- يتصل بقاعدة بيانات Neon عند توفر الإنترنت
- يمكن توزيعه على أجهزة النادي دون متصفح

### على السحابة (Vercel)
- الرابط: https://aladine-pool-manager.vercel.app
- قاعدة البيانات: Neon PostgreSQL
- يعمل من أي متصفح، أي جهاز، أي مكان

## المزامنة

- **تلقائية**: كل 5 دقائق عند الاتصال
- **عند عودة الإنترنت**: فوراً
- **يدوية**: زر المزامنة في الرأس
- **حل التعارضات**: آخر تعديل يفوز (last-write-wins)

## مؤشر الحالة (SyncIndicator)
- 🟢 متصل + متزامن
- 🟡 جاري المزامنة...
- 🔴 غير متصل (أوفلاين)
- يعرض وقت آخر مزامنة

## الملفات الرئيسية

| الملف | الدور |
|------|------|
| `public/sw.js` | Service Worker — كاش التطبيق للعمل أوفلاين |
| `public/manifest.json` | PWA manifest — يجعل التطبيق قابلاً للتثبيت |
| `src/lib/local-db.ts` | طبقة IndexedDB — تخزين محلي للبيانات |
| `src/lib/sync.ts` | محرك المزامنة — bidirectional sync |
| `src/components/sync-indicator.tsx` | مؤشر حالة المزامنة في الواجهة |
| `electron/main.ts` | نقطة دخول تطبيق سطح المكتب |

## التثبيت على الهاتف

### Android (Chrome)
1. افتح الرابط في Chrome
2. القائمة (⋮) → "تثبيت التطبيق"
3. تظهر أيقونة على الشاشة الرئيسية

### iOS (Safari)
1. افتح الرابط في Safari
2. زر المشاركة → "إضافة إلى الشاشة الرئيسية"
3. تظهر أيقونة على الشاشة الرئيسية

### كمبيوتر (Chrome/Edge)
1. افتح الرابط
2. أيقونة التثبيت في شريط العنوان
3. التطبيق يفتح في نافذة مستقلة
