## 📋 دليل التثبيت

### للويب
1. افتح: https://aladine-pool-manager.vercel.app
2. سجّل الدخول كـ admin
3. ابدأ الاستخدام

### لسطح المكتب (Windows)
1. حمّل مجلد `dist/` أو `RCS-Club-1.0.0-Portable.zip`
2. استخرج الملفات
3. شغّل `Setup.bat` كمسؤول — أو — شغّل `RCS Club.exe` مباشرة
4. التطبيق سينشئ قاعدة بيانات SQLite تلقائياً

### للبناء من المصدر
```bash
git clone https://github.com/flashnet20dz/rcs-swim.git
cd rcs-swim
npm install
npx prisma generate
npm run build        # للويب
build-setup.bat      # لسطح المكتب
```

## 📋 دليل الاستخدام

### البداية
1. سجّل الدخول كـ admin
2. اذهب للإعدادات → الترويسة الموحدة — صمّم ترويسة تقاريرك
3. اذهب للإعدادات → أنواع الاشتراك — أضف أنواع ناديك
4. ابدأ بإضافة المنخرطين

### الاستيراد من Excel
1. اذهب لتبويب "الاستيراد"
2. ارفع ملف Excel (يجب أن يحتوي أعمدة: اللقب، الاسم، تاريخ الميلاد، الجنس)
3. راجع البيانات في واجهة المراجعة الكاملة
4. صحّح الأخطاء أو استبعد الصفوف
5. اضغط "استيراد جميع الصفوف الصالحة"

### التقارير
1. اذهب لتبويب "التصدير" → مركز التقارير
2. اختر التقرير (15 تقرير متاح)
3. استخدم الفلاتر + البحث
4. صدّر PDF/Word/Excel أو اطبع

### النسخ الاحتياطي (Desktop فقط)
1. الإعدادات → سطح المكتب → النسخ الاحتياطي
2. اختر: يدوي الآن / استعادة / تلقائي
3. للنسخ التلقائي: فعّل + اختر التكرار (يومي/أسبوعي/شهري)

## 📋 دليل النسخ الاحتياطي

### على Desktop
- **المسار الافتراضي:** `%APPDATA%\RCS Club\backups\`
- **تلقائي:** يومي عند بدء التشغيل (إذا مُفعّل)
- **يدوي:** الإعدادات → سطح المكتب → نسخة احتياطية الآن
- **الاستعادة:** الإعدادات → سطح المكتب → استعادة نسخة

### على الويب
- البيانات محفوظة على Neon PostgreSQL (سحابي)
- النسخ الاحتياطي عبر Vercel + Neon
- يمكن تصدير البيانات عبر تبويب "النسخ الاحتياطي"

## ⚠️ ملاحظات

1. **Setup.exe النهائي**: يتطلب بناء على جهاز Windows أو Wine. استخدم `build-setup.bat` على Windows.
2. **Wine غير متوفر** في بيئة التطوير الحالية، لذلك تم إنشاء `Setup.bat` كبديل يعمل كـ installer كامل.
3. **قاعدة البيانات على Desktop** تُنشأ تلقائياً عند أول تشغيل (SQLite).
4. **التطبيق يعمل Offline** بالكامل على Desktop.
